# Modelling Lava Flow with Barriers using rascommander

This repository is written by [Quynh Phuong Le](mailto:phuongqle.data@gmail.com) to showcase the work on adapting **HEC-RAS**, a GIS-based hydraulic flow simulation software, to model lava flow with barriers. For more information about the lava flow modelling with barriers project, please contact  [Dr Edward Hinton](https://blogs.unimelb.edu.au/edward-hinton/#tabmain)

The instructions in this repository are based on **rascommander**, a Python library for automating HEC-RAS workflows, and were written in October 2025. As rascommander continue to evolve, some functionality described here may have changed in newer versions.

## Repository structure
- HEC-RAS Manage Lava Barrier.ipynb: Gives basic instructions on how to get started with rascommander and how to manage SA/2D connection to represent lava barriers.

- HEC-RAS Parallel Execution and Extract data.ipynb: Gives instructions on how to execute multiple simulation scenarios in parallel and to extract simulation results at points along the barriers, such as flow velocity, water/lava surface elevation.
