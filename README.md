# Modelling Lava Flow with Barriers using rascommander

This repository showcases my work on adapting **HEC-RAS**, a GIS-based hydraulic flow simulation software, to model lava flow with barriers. The instructions use **rascommander**, a Python library to automate HEC-RAS operations.

## Repository structure


- Create and manage **Storage Area (SA) / 2D Area connections** in HEC-RAS to represent lava barriers.
- Automate HEC-RAS simulations through Python.
- Execute multiple simulation scenarios in parallel.
- Extract simulation outputs at specified locations along the barriers, including flow velocity, water/lava surface elevation, and other relevant hydraulic variables.
- Analyse the effectiveness of different barrier configurations for lava flow mitigation.

The repository contains the scripts, workflows, and documentation developed during this project, providing a reproducible framework for automating HEC-RAS simulations and analysing barrier performance.
